# Repository Structure

The structure of this project is as follows:

```tree
/data-science-portfolio/                # Root directory
├── README.md                          # Project overview and basic information
├── LICENSE                           # License file for the project
├── requirements.txt                  # Python package dependencies
├── vs_code_setup.md                 # VS Code setup guide
├── notebooks/                       # Jupyter notebooks directory
│   ├── 01_eda.ipynb               # Exploratory Data Analysis
│   ├── 02_baseline.ipynb         # Baseline model implementation
│   └── 03_improvements.ipynb    # Future improvements and experiments (if needed)
├── src/                         # Source code directory
│   ├── data/                  # Data directory
│   │   └── README.md        # Data documentation and description
│   ├── models/              # Machine learning models directory
│   └── utils/              # Utility functions
│       ├── __init__.py    # Package initialization
│       ├── data_loader.py # Data loading utilities
│       └── preprocessor.py # Data preprocessing utilities
├── tests/                  # Unit tests directory
└── docs/                  # Documentation directory
    ├── motivation.md     # Project motivation and goals
    ├── design.md        # System design documentation
    ├── repository_structure.md  # Project structure explanation
    └── api_documentation.md   # API documentation
```
